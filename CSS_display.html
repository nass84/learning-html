Changing how the block appears

More information

Using block element

This technique is often used on links in order to increase their clickable zone, which can be easily evaluated by setting a background color.

.menu a{ background: red; color: white; display: block;}

Using Inline element

This can make lists appear to the side of each other instead of underneath

.menu li{ display: inline;} - CSS
ul class="menu" - html

Using Display none

This can remove it from your website

.gone-baby-gone{ display: none;}

Using hidden

.hollow-man{ visibility: hidden;} - hides the class

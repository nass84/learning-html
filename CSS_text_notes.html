CSS Text

    Fonts are grouped in 5 generic families:
  • serif
  • sans-serif
  • monospace
  • cursive
  • fantasy

You can apply it to whole html document with:
body{ font-family: sans-serif;}

Web-safe fonts

To make your web page the same on all computers use the name of the font.

body{ font-family: Arial;}
    There are 9 Web-safe fonts
  • Arial
  • Arial Black
  • Comic Sans MS
  • Courier New
  • Georgia
  • Impact
  • Times New Roman
  • Trebuchet MS
  • Verdana

You can define fallback values by writing a list of font families: body{ font-family: Arial, Verdana, sans-serif;} It will look fr Arial first, then Verdana etc Use a generic family as last value.

Font Style Tags

p{ font-size: 16px;} - change font size
h2{ font-style: italic;} - change to italic, default is normal
h2{ font-weight: bold;} - change font to bold

Line Spacing

for body text, a line height of 1.5 times the size of the text is recommended. for headings, a line height of 1.2 is recommended

body{ font-size: 16px; line-height: 1.5;

CSS Font Shorthand

You can use shorthand to style the text all in one go


body{ font: italic small-caps bold 16px/1.5 Arial, sans-serif;}

Must be in that order

Other Text Alterations

body{ text-align: left;} - aligns text to left, can also use right center justify

.deleted{ text-decoration: line-through;} - text-decoration puts a line through the text can also use overline underline line-through
blockquote{ text-indent: 30px;} - Adds space before the first word, use padding for whole block of text
h1{ text-shadow: 0 2px 5px rgba(0,0,0,0.5);} - add shadow to the text

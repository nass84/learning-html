Backgrounds are only applied on the targeted element. But considering most HTML elements have a transparent background, applying a background on the body will look like itâ€™s applied on all elements.


body{ background: #f2eee9;} - whole background filled with a plain background color

body{ background-image: url(images/diagonal-pattern.png);} - applies an image to background (may need to be resized>

The HTML element is for images that are part of the content, while CSS background images are purely decorative.

Background image tags

body{ background-image: linear-gradient(white, grey);} - applies a gradient to background

body{ background-position: right bottom;} OR body{ background-position: center 20px;} - changes position of the background image and size

By default, a background image will repeat itself indefinitely. You can choose to make it repeat only horizontally, only vertically, or not at all.

body{ background-repeat: repeat-x;} /* Only horizontally */
body{ background-repeat: repeat-y;} /* Only vertically */
body{ background-repeat: no-repeat;} /* The background image will only appear once */

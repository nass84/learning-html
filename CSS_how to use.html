CSS

CSS stands for Cascading Style Sheets.
You can either style it in the HTML element
Or you can use style tag in the head of your html doc
Preferred method You can write CSS in a seperate file using .css extenstion and link to HTML using link tag.

CSS is a 3-part process: the selector defines who is targeted, which HTML element(s) the property defines what charateristic to alter the value defines how to alter that charateristic
We can have multiple selectors, multiple properties, and sometimes (but rarely) multiple values.

Generic tag selectors

Targeting generic HTML tags is easy: just use the tag name.

class

You can use class to style a certain paragraph or html element class="classname" For the CSS you have to put.classname

id

For id you use a # in the CSS #idexample and put id="idexample" in html, this over rides the paragraph rules

Combining selectors

If you want to combine selectors you can add a . between them i.e em.date { color: blue; } The em.date combines: a tag selector em a class selector .date It will only apply to the em class="date" not any other .date or em

Hierachy

A space in a selector defines a ancestor/descendant relationship. Let’s say we want the links in our header to be in red: header a { color: red; } This can be read from right to left as: “Select all a elements that are within a header element”. This will prevent all other links (that aren’t in the header) from being affected.

Pseudo-class selectors - Hover

HTML elements can have different states. The most common case is when you hover over a link. It’s possible in CSS to apply a different style when such an event occurs. Pseudo-class selectors are attached to usual selectors and start with a colon :: a { color: blue; } a:hover { color: red; }
